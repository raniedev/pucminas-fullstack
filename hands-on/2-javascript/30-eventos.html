Eventos

jQuery também oferece uma alternativa à criação de eventos. No entanto, é importante saber que o evento não é criado diretamente sobre o element HTML, mas sobre o conjunto resposta da seleção.

$('#botao').click(() => alert('Olá mundo'));

jQuery permite o tratamento de todos os eventos JavaScript, como, por exemplo:

  • click()
    clique do mouse
  • dblclick()
    duplo clique
  • hover()
    entrada (e saída) do ponteiro do mouse sobre elementos
  • mousemove()
    movimento do mouse
  • change()
    alteração do valor do campo
  • focus()
    foco atribuído ao campo
  • keydown()
    tecla pressionada no campo
  • submit()
    formulário enviado
  • etc...

Eventos especiais

  • on(evento, [seletor], [dados], função)
    Associa uma função a um evento para os elementos desta seleção. Opcionalmente, a função pode ser associada a um descendente. Os dados também são opcionais;
  • off(evento)
    Remove o tratamento de um evento de um elemento;
  • one(evento, [dados], função)
    semelhante ao on(), mas que só é executado uma única vez
  • trigger(evento, [dados])
    força a execução de um determinado evento de algum elemento

O objeto evento

A função do evento recebe um objeto com vários dados, como esses:

Função Descrição
type tipo de evento (p.ex., 'click')
target elemento que disparou o evento
data dados passados para função que tratará o evento
pageX, pageY coordenadas do mouse quando o evento foi disparado
timeStamp momento em que o evento ocorreu
preventDefault() evita que o navegador executa a ação padrão
stopPropagation() evita a propagação do evento para os elementos ancestrais
