Módulos

Na medida em que a aplicação começa a crescer, a gente precisa organizá-la de uma forma diferente, separando os códigos por funcionalidade e em arquivos específicos. Esses arquivos são os módulos.

Os módulos podem ser carregados de acordo com a necessidade, diminuindo o tempo inicial da aplicação e reduzindo o uso de memória.

Módulo

Um módulo é nada mais do que um arquivo de script em que tudo(variáveis, constantes, funções, ...) que puderem ser usadas externamente tem um nome exportado.

const PI = 3.14159;
export class Circulo {
    constructor(r) {
        this.raio = r;
    }

    perimetro() {
        return 2 * PI * this.raio;
    }

    area() {
        return PI * Math.pow(this.raio, 2);
    }
}
