JavaScript Object Notation

JSON é um formato para troca de dados entre sistemas baseado em JavaScript.

Um objeto JSON se parece muito com um objeto JavaScript, mas com os nomes das propriedades entre aspas.

Covertendo um objeto JSON para texto

Algumas vezes, precisamos transformar um objeto JSON em uma string para enviarmos por um formulário, pela rede ou para armazenarmos localmente.

let usuario = {"nome": "João", "idade": 35};
let str_usuario = JSON.stringify(usuario);

Convertendo de volta para objeto

Ao recebermos de uma resposta HTTP ou do armazenamento local, um objeto que está na forma de uma string, podemos reconstruí-lo da seguinte forma:

let = usuario = JSON.parse(str_usuario);
