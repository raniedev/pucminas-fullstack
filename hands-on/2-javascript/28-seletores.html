Seletores

jQuery oferece uma série de funções que facilitam a leitura e a manipulação dos elementos selecionados da página.

Os seletores não retornam os elementos HTML da página, mas um conjunto de objetos da própria jQuery que atendam aos critérios dos seletores. Esses critérios são os mesmos da CSS.


$('#ola').click() => alert("Olá mundo!"));
...
<button id="ola">Olá mundo.</button>

Comparação entre JavaScript e jQuery

JavaScript

  • document.getElementById('lista1')
  • document.querySelector('header < h1')
  • document.getElementByTagName('p')
  • document.getElementById('botao').addEventListener('click', ()=>alert("Olá mundo!"))

jQuery

  • $('#lista1')
  • $('header > h1')
  • $('p')
  • $('#botao').click(()=>alert("Olá mundo!"))
Seletor Aplicação
* Seleciona todos os elementos
element Seleciona os elementos específicos
anc desc Seleciona os descendentes especificados de um elemento (ancestral e descendentes)
parent > child Seleciona um elemento com o determinado identificador
#id Seleciona um elemento com o determinado identificador
class Seleciona os elementos de uma determinada classe
element.class Seleciona os elementos especificados de uma dada classe
prev + next Seleciona os elementos especificados que são precedidos por um determinado elemento, e que tenham o mesmo pai
prev ~ siblings Seleciona os elementos especificados que são precedidos por um determinado elemento, e que tenham o mesmo pai
e1, e2, e3 Seleciona os elementos especificados
