Expressões de Função

Uma expressão de função é uma função (geralmente anônima) que pode ser atribuída a uma variável como uma expressão qualquer.

E essa função pode ser invocada da mesma forma que uma função tradicional

Invocacao Imediata

Mas se ela é uma expressão, ela pode ser executada diretamente, sem a necessidade de ser armazenada em uma variável.

Para isso, o compilador apenas precisa saber que não se trata de uma declaração de função tradicional (colocando a expressão entre parênteses).

(function(nome) {
    console.log(`Olá ${nome}`);
})('Marcos');

Isso é chamado de: IIFE - Immediately Invoked Function Expression

Atenção

Se atribuirmos uma IIFE a uma variável, o que realmente será armazenado é o resultado do processamento da função.

let ola = (function (nome) {
    return `Olá ${nome}`;
})('Marcos');

console.log(ola); // Olá Marcos
console.log(typeof ola); // string
ola(); // TypeError
