Manipulação da Página

Alteração do conteúdo

Uma das principais operações que fazemos com jQuery é a alteração da página, seja do seu conteúdo, da sua formatação ou do seu comportamento.

  • html()
    Retorna o conteúdo HTML do primeiro elemento do conjunto
  • html(novo conteúdo)
    Define o conteúdo HTML de cada elemento do conjunto
  • text()
    Retorna o conteúdo textual do primeiro elemento do conjunto
  • text(novo texto)
    Define o conteúdo textual de cada elemento do conjunto

Acesso aos atributos

  • attr(nome)
    Recupera o valor de um atributo. Se não existir, retorna undefined
  • attr(nome, valor)
    Atribui o valor ao atributo nome
  • attr(propriedades)
    Define uma série de atributos para o conjunto de elementos usando a notação de objetos(pares de nomes e valores dos atributos)
  • attr(nome, fn)
    Define uma função para calcular o valor do atributo nome em cada elemento do conjunto
  • removeAttr(nome)
    Remove o atributo nome de cada elemento do conjunto

Formatação

  • css(propriedade)
    Retorna o valor da propriedade CSS do primeiro elemento do conjunto
  • css(propriedade, valor)
    Define o valor da propriedade CSS para todos os elementos do conjunto
  • css(propriedades)
    Define várias propriedades para os elementos conjunto usando a notação de objetos(pares de nomes e valores das propridades CSS)

Manipulação de Classes

  • addClass(classe)
    Adiciona uma ou mais classes a cada elemento do conjunto
  • hasClass(classe)
    Informa se a(s) classe(s) está(ão) presente(s) em pelo menos um elemento do conjunto
  • removeClass(classe)
    Remove a(s) classe(s) dos elementos do conjunto
  • toggleClass(classe)
    Adiciona a classe se não estiver presente, remove se estiver
  • toggleClass(classe, booleano)
    Adiciona a classe se o valor for verdadeiro, remove se o valor for falso
