jQuery

jQuery é uma biblioteca JavaScript que simplifca o percurso de um documento HTML, o gerenciamento de eventos, as animações e as interações Ajax para o desenvolvimento rápido de alicações Web.

Vantagens

  • Acelera o processo de desenvolvimento
  • Operações que exigiram vários comandos são simplificadas
  • É compatível com todos os navegadores (I)*
  • É extensível através de plugins (II)*
I - Checar compatibilidade
II - Checar plugins

Instalação

  • jQuery pode ser usada de duas formas:
  • Por meio de um arquivo jquery-X.X.X.min,js instalado localmente. Os arquivos podem ser baixados em link
  • Poder meio do CDN, cujos arquvios podem ser encontrados em link
  • Cada arquivo possui uma versão X.X.X específica.
  • O código deve ser usado na sua versão minificada (compactada).

Exemplo de uso

$(document).ready(function(){
    $('a').click(function(evento){
        alert('O link não funciona mais.');
        evento.preventDefault();
    });
});
