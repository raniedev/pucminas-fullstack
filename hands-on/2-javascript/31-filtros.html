Filtros

Um filtro é uma forma de reduzir o conjunto de elementos selecionads àqueles que atendam a uma determinada condição.

$('tr').filter(':even').css('background-color', 'lightgrey');

Esse filtro retorna apenas os elementos pares (1 a cada 2) de uma determinar seleção.

Função Descrição
filter() Reduz o conjunto para aqueles que atendem a um seletor ou passem em uma função de teste
is() Testa se pelo menos um objeto atende ao seletor e, nesse caso, retorna verdadeiro
not() Elimina do conjunto os elementos que se enquadrem no segundo seletor
has() Reduz o conjunto àquele elemento que possuam um determinado descendente
eq() Reduz o conjunto para apenas um elemento do índice especificado
first() Reduz o conjunto ao seu primeiro elemento
last() Reduz o conjunto ao seu último elemento
slide() Reduz o conjunto a um subconjunto especificado por uma faixa de índices

Transformação

Também é possível transformarmos um conjunto em outro, por meio do método map()

//Retorna os ids dos elementos selecionados
let listaIDS = $(':input').map(function(){
    return $(this).attr('id');
}).get();

Essa função retorna o valor do atributo id de cada elemento, criando um novo vetor de IDs. Finalmente, usamos o método get() para retornar o resultado como um objeto JavaScript comum.
