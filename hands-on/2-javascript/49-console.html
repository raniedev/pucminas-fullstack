Mensagens no Console

Apesar de estarmos sempre usando o método log() do objeto console, ele oferece muitas outras alternativas mais específicas:

  • console.log("mensagem")
  • console.info("informação")
  • console.debug("depuração")
  • console.warn("alerta")
  • console.error("erro")

Dados nas mensagens

Podemos inserir dados na mensagem, usando esses marcadores:

  • %d - número
    console.log('Temperatura: %d graus', 37);
  • %s - string
    console.log('Olá %s. Bem-vindo', 'Marcos');
  • %o - objeto
    let usuario = {nome: 'Marcos', idade: 51};
    console.log('Seus dados: %o', usuario);
  • %c - CSS
    console.log('%cAtenção', 'color: red; background-color: yellow;');

Vetores

Vetores podem ser impressos na forma de uma tabela:

let produtos = [
    {nome: 'TV', preço: 2190.0},
    {nome: 'Geladeira', preço: 1700.0},
    {nome: 'Microondas', preço: 519.0}
];
console.table(produtos);
(index) nome preço
0 "TV" 2190
1 "Geladeira" 1700
2 "Microondas" 519

Grupos

Vetores podem ser impressos na forma de uma tabela:

console.group();
console.log('Grupo 1');
console.group();
console.log('Subgrupo');
console.log('Subgrupo também');
console.groundEnd();
console.groundEnd();
console.group();
console.log('Grupo 2');
console.groupEnd();

Contadores

Podemos ter 1 ou mais contadores paralelos

console.count('a');
console.count('a');
console.count('a');
console.count('b');
console.count('a');
console.count('b');

Testes

Também podemos testar algumas condições e só registrar uma mensagem(de erro) se as condições falharem

for(let i = 0; i < 5; i++) {
    console.assert(i % 2 == 0, '%d não é par', i);
}

Pilhas de chamadas

Às vezes, acompanhar que função estão chamando a função que deu erro pode ser um pouco complicado. Este comando ajuda a identificar a pilha de chamadas

function fn1() {
    fn2();
}

function fn2() {
    consol.trace('Chamada na fn2');
}

fn1();
fn2();

Cronômetro

Também é possível medir o tempo de uma operação.

console.time('T1');
let total = 0;
for (let i = 0; i < 10000; i++) {
    total += i;
}
console.timeEnd('T1');
