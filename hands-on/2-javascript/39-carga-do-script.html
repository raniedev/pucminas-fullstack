Carga dos Scripts

async/defer

Os atributos async e defer são usados para determinar a forma de carga do script e o momento da sua execução, especialmente quando isso leva muito tempo.

  • Sem atributos - O script é carregado e executado imediatamente.
    <script src="script.js"></script>
  • async - O script é carregado paralelamente e executado logo que esteja pronto.
    < async src="script.js"></script>
  • defer - O script é carregado paralelamente e executado somente quando o DOM estiver pronto.
    <script defer src="script.js"></script>
