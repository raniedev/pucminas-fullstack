Canvas

A api canvas transforma a página em uma área em que podemos desenhar livremente.

Esse desenho pode ser feito por meio de funções básicas 2d ou 3d, como os círculos, retângulos, linhas poligonais e muito mais.

JavaScript

O retângulo é um das formas básicas da API

  • fillRect(x,y,width,height)
    desenha um retângulo preenchido
  • clearRect(x,y,width,height)
    limpa uma área (preto transparente)
  • strokeRect(x,y,width,height)
    desenha o contorno de um retângulo

Caminhos (path)

O caminho é a segunda forma básica da API

O desenho é feito a partir do rastro do cursor

O controle do caminho pode ser feito com as funções:

  • beginPath() - inicia um novo caminho;
  • closePath() - conecta o ponto atual ao início
  • moveTo() - move o ponto sem traçar
  • stroke() - colore a linha
  • fill() - enche a área do caminho
Seu navegador não suporta Canvas.

Outras formas para desenho:

  • linhas lineTo(x, y)
  • arcos arc(x, y, raio, angulo_inicial, angulo_final, antihorario?)
  • curva quadrática quadraticCurveTo(ctr_x, ctr_y, x, y)
  • curva de bézier bezierCurveTo(ctr1x, ctr1y, ctr2x, ctr2y, x, y)

Cores

Propriedades do contexto

  • fillStyle - cor de preenchimento
  • strokeStyle - cor do traço
  • globalAlpha - opacidade (de 0.0 a 1.0)

Valores de cores

  • orange, black, green etc.
  • #ffa500
  • rgb(255, 165, 0)
  • rgba(255, 165, 0, 0.5)

Estilos de linha

Propriedades do contexto

  • lineWidth espessura da linhas
  • lineCap tipo de ponta (butt, round ou square)
  • lineJoin
    tipo de junção (round, bevel, miter)
  • miterLimit
    limite da distância da conexão dos pontos externos da linha à conexão dos pontos internos com a junção do tipo miter(as pontas das conexões podem ficar maiores ou menores)
